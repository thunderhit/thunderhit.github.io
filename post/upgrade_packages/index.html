如何批量更新已安装的库

方法1: pip

命令行切换到桌面

cd desktop

在桌面生成requirements.txt，该文件主要记录本地安装的包及其版本号

pip freeze > requirements.txt

在桌面找到requirements.txt，将该txt文件中 “==” 改为 “>=”，命令行执行

pip install -r requirements.txt --upgrade

方法2: pip-review

首先在电脑中安装pip-review包

pip install pip-review

Pip-review有两种更新方式

更新方式
自动(批量) pip-review --auto
逐个手动确认 pip-review --interactive
