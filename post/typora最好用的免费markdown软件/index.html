N级标题

列表

有序列表

ordered list:
1.  Red
2. 	Green
3.	Blue

ordered list:

  1. Red

  2. Green

  3. Blue

注意.之后留一个空格再书写内容

无序列表

un-ordered list:
- Red
- Green
- Blue

效果如下

un-ordered list:

  • Red

  • Green

  • Blue

注意.之后同样留一个空格再书写内容



代码块

一般情况下，word等office软件不支持代码高亮，且存储代码容易乱行。以typora为首的markdown语法完美支持各种代码，这里以Python代码为例

​```python
import requests
url = 'https://www.baidu.com/'
resp = requests.get(url)
​```

效果如下

import requests
url = 'https://www.baidu.com/'
resp = requests.get(url)

mermaid图表

Typora中可以绘制流程图、序列图、状态图、甘特图、饼形图、类图等，这里以流程图为例

​```mermaid
graph LR
	A-->B
	B-->C
	C-->D
​```

效果

graph LR
	A-->B
	B-->C
	C-->D
